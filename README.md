# Heart-Rhythm-Simulator-Diagnostic-Tool

It is designed to bridge the gap between digital signal processing and clinical diagnostics. It allows users to input heart rate data (BPM) via a Graphical User Interface (GUI) to generate high-fidelity, simulated Electrocardiogram (ECG) waveforms. The tool automatically classifies heart rhythms based on standard clinical thresholds, such as Tachycardia and Bradycardia.

## How it works
1. input the BPM and click analyze
2. it will show a graph
